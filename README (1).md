# МобиФикс — Ремонт телефонов с выездом

Сайт-визитка для сервиса выездного ремонта смартфонов.

## Возможности

- Заявка на выезд мастера
- Отслеживание статуса заявки по номеру
- Каталог услуг с ценами
- Адаптивный дизайн (мобильные + ПК)

## Технологии

- HTML5 / CSS3 / JavaScript
- Без фреймворков — один файл index.html

## Запуск локально

Просто откройте `index.html` в браузере.

## Деплой на Vercel

1. Загрузите репозиторий на GitHub
2. Зайдите на [vercel.com](https://vercel.com)
3. Подключите этот репозиторий
4. Нажмите Deploy

## Контакты

Замените в `index.html` номер телефона и адрес на свои данные.
